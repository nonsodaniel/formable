# Formable

Formable is a customizable form builder that allows users to generate forms based on their personal preferences and requirements. It simplifies the process of creating dynamic, reusable forms without manually coding each form from scratch. The project demonstrates scalable frontend architecture, dynamic UI generation, and modern full-stack integration using Next.js and Prisma.

## Features

- Dynamic form generation based on user preferences  
- Customizable form fields and structure  
- Modular and scalable component architecture  
- Fast and responsive user interface  
- Database integration using Prisma  

## Technologies Used

- Next.js  
- React.js  
- TypeScript  
- Prisma (ORM)  
- Node.js  
- ESLint  

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (v18 or newer recommended)
- npm, yarn, pnpm, or bun
- A database supported by Prisma (PostgreSQL, MySQL, SQLite, etc.)
